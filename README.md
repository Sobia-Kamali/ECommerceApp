# E-Commerce App

A Java-based desktop E-Commerce Application built with Swing, providing a simple, interactive shopping experience with user authentication, product browsing, and cart management.

---

Features

User Authentication
  - Sign-up and login functionality.
  - Role-based users (Admin/User).

- Product Listing
  - Products displayed as cards with images, names, and prices.
  - Category filter using a dropdown menu.
  - Search functionality for easy product lookup.

- Shopping Cart
  - Add or remove products from the cart.
  - Cart summary with total price.
  - Orders restricted to logged-in users.

- Ratings System
  - Users can rate products (optional based on your implementation).

- Responsive UI
  - Scrollable product view (vertical scrolling supported).
  - Intuitive, user-friendly interface.
